# TETRIS

### Environment
- RedHat Linux 6.2, GCC compiler

### Screenshot
![playing-tetris.png](images/playing-tetris.png)

### Feature
* ↑: Rotate
* ←→: Move left/right
* ↓: Down (wait)
* Space: Down (without wait)
* Do not rotate when blocked
* Random blocks
* Score
* High score
* Shadow block (show block down position)
* Pause game
* Level
* Elapsed time
